What is dom? What's the purpose of dom?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.
It defines the logical structure of documents.

How to element select from dom?

There are many different way to select element:
1. getElementsByClassName – search element by class name.
2. querySelectorAll – returns elements that match the specified selector
3. getElementById – search element by element_id.
4. getElementsByTagName – search element by tag name (e.g., span, div)
5. getElementsByName – search element by name attribute.
6. querySelector – returns the first element that matches the specified selector

What is event bubble?

Event bubbling and capturing are two ways of event propagation in the HTML DOM API, when an event occurs in an element inside another element, and both elements have registered a handle for that event. The event propagation mode determines in which order the elements receive the event.
